# Retail Sales Dashboard Using Microsoft Excel

## 📊 Project Overview

This project demonstrates the complete data analytics workflow by transforming a raw retail sales dataset into an interactive business dashboard using Microsoft Excel. The project involved data cleaning, transformation, analysis, and visualization to generate meaningful insights that support business decision-making.

The dashboard enables users to monitor sales performance, analyze customer purchasing behavior, evaluate product performance, and identify sales trends through interactive reports.

---

## Objectives

- Clean and prepare raw retail sales data.
- Handle missing and inconsistent data.
- Transform data using Power Query.
- Analyze sales performance using PivotTables.
- Build an interactive dashboard with PivotCharts and slicers.
- Generate actionable business insights.

## Data Cleaning Process

The original dataset contained several data quality issues that were resolved using Power Query and Excel.

The cleaning process included:

- Handling missing values
- Correcting inconsistent data entries
- Filling missing *Price per Unit* values using item-price relationships
- Verifying calculated fields
- Formatting dates and numeric columns
- Assigning appropriate data types
- Removing inconsistencies
- Preparing the dataset for analysis
